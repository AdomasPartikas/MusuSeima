# musu-seima.lt

This will be a personal project for me and my family. The goal is to create a personal, not complex website that will have a singular button which will link to a google drive folder called "Mūsų Šeima". The aim of the drive is a photo and a memory archive. 

Additionally, scrolling down in the page after the button would be a memory spotlight with photos accompanied by names, year they were taken and a short description.
The spotlight does not to be large, but can be interactive in a fun way.

The main goal is to make the website easy to navigate for all ages.
